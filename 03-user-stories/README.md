# User stories — CV Instant Pro (MVP 7 jours)

Règles :
- Chaque US est **atomique**, **testable**, et livrable indépendamment.
- Estimations : Fibonacci (1, 2, 3, 5, 8)
- "Done" = critères d’acceptation + scénarios de test validés.

- Sprint 2 (J3.5–J7) : Worker Playwright + rendu PDF + download + notifications + déploiement
